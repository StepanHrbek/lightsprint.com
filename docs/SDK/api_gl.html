LightsprintGL

Overview

LightsprintGL consists of 2 major components

  1. integration of GI solvers with OpenGL
    • implemented in rr_gl::RRSolverGL
  2. extensible high performance OpenGL renderer
    • implemented in rr_gl::Renderer

API

Namespace: rr_gl

Header: Lightsprint/GL/RRSolverGL.h

  • rr_gl::RRSolverGL integrates GI solvers with OpenGL 2.0

Header: Lightsprint/GL/Renderer.h

  • rr_gl::Renderer is a plugins based renderer

Other headers/classes:

  • rr_gl::Texture is OpenGL texture, extension of RRBuffer needed for realtime rendering
  • rr_gl::Program is GLSL program
  • rr_gl::UberProgram is GLSL program with preprocessor parameters changeable at runtime
  • rr_gl::UberProgramSetup is set of parameters for our ubershaders UberShader.vs and UberShader.fs
  • rr_gl::RealtimeLight is an extension of RRLight, structures needed for realtime GI rendering
  • rr_gl::FBO lets you render to textures
